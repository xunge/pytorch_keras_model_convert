## 借助 onnx 进行 keras 与 pytorch 模型间转换

测试环境1：RTX 3090
- tensorflow-gpu==2.4.0
- tensorflow-addons==0.16.1
- pytorch==1.7.0
- onnx==1.11.0
- cuda==11.0
- cudnn==8.0.5

测试环境2：RTX Titan
- tensorflow-gpu==2.2.0
- tensorflow-addons==0.10.0
- pytorch==1.7.1
- onnx==1.11.0
- cuda==10.1
- cudnn==7.6.5

以上环境均可测试通过。
